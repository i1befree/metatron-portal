Query작성 도움말

  • 변수 유형(Namespace)은 text, number, date, select, array, datetime 6가지를 지원한다.
  • Query작성 시 구성요소(Attribute)는 Namespace, Label, Arguments, Description 4가지로 구성되며
    아래와 같은 형식을 참고해 작성한다. ${Namespace:Label('Arguments')|'Description'}
변수 유형 Query작성 예시
text ${text:abc('def')|'변수입력'}
number ${number:cnt('123')|'숫자입력'}
select ${select:성별('남', '여')|'성별선택'}
array ${array:arr('a', 'b', 'c')|'a,b,c 배열을 입력'}
date ${date:startDate('yyyyMMdd')|'날짜입력'}
datetime ${datetime:startTime('yyyyMMdd','HH:mm:ss')|'시간입력'}
